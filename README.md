## :key: Getting Started

```bash
# clone the repo:
git clone https://github.com/Regis-Oliveira/Form-exercise-SH.git

# open the project:
cd Form-exercise-SH

# install the dependencies:
npm install
# or
yarn install

#execute
npm run dev
# or
yarn dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
